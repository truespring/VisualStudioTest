연산자

1. 산술연산자 (+, -, *, /, %)
2. 증감연산자 (++, --)
- 증가연산자(++) : 변수의 앞 또는 뒤에 붙여 사용, 전위, 후위
- 감소연산자(--) : 변수의 앞 또는 뒤에 붙여 사용, 전위, 후위
3. 비교(관계)연상자 : <, <=, >, >=, ==, != 값만 비교하여 결과를 알려준다
- ===, !== : 자바스크립트에서 추가된 연산자 값과 타입을 한꺼번에 비교하여 결과를 알려준다
- 결과가 논리값 형태로 표현, true 또는 false
- 우선순위 : (>, >=, <, <=) 연산자가 (==, !=) 보다 우선이다.
- 우선순위 : (==, !=) 연산자가 (===, !==)보다 우선이다.
4. 논리연산자 : &&(And), ||(Or), !(논리 Not)
- 논리연산자의 결과는 논리값(true, false 둘 중의 하나)
- 비교연산자가 논리연산자보다 우선이다. 비교 > 논리
- 우선수위 : 논리And가 논리Or보다 우선이다.
- 우선순위 : ! > && > ||
5. 대입연산자 : =
- 복합대입연산자(할당연산자, assignment) : +=, -=, *=, /=, %=
